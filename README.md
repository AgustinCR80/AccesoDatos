# AccesoDatos
Asignatura Acceso de Datos
Explicación del menu:
  1 Añadir fruta. 
    - Se importa el archivo .txt donde se guarda el inventario de las frutas. Si éste no existe lo crea. Añadimos nuevas frutas en un ArrayList.

  2 Ver últimas frutas añadidas.
    - Se muestran por pantalla las frutas añadidas antes de que sean guardadas en el inventario
