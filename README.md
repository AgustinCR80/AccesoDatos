# AccesoDatos
Asignatura Acceso de Datos
Explicación del menu:
  1 Añadir fruta. 
    - Se importa el archivo .txt donde se guarda el inventario de las frutas. Si éste no existe lo crea. Añadimos nuevas frutas en un ArrayList.

  2 Ver últimas frutas añadidas.
    - Se muestran por pantalla las frutas añadidas antes de que sean guardadas en el inventario, una vez guardadas con la opción 4 del menu dejaran de ser visibles con esta opción. 

  3 Mostrar inventario.
    - Se importa el archivo .txt donde se guarda el inventario de las frutas. Si éste no existe lo crea y al estar vacío nos lo indica. 

  4 Guardar inventario.
    - Añade las últimas frutas añadidas al inventario, si no se añade ninguna nos lo indica.

  5 Salir.
    - Salir del menu.
